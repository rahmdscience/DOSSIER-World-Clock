# 🌍 Midnight Editorial World Clock

A premium **Midnight Editorial Dashboard** styled web application for tracking global time, managing alarms, and performing time-based utilities.

## ✨ Features

- 🌎 Multi Timezone World Clock
- ⏱ Stopwatch (with lap tracking)
- ⏳ Countdown Timer
- ⏰ Alarm System with sound
- 🔁 Timezone Converter
- 🎯 Real-time UTC synchronization
- 🎨 Clean Editorial UI Design

## 🎨 Design Style

This project uses a **Midnight Editorial Dashboard** style:

- Dark premium interface
- Minimal and professional layout
- Typography-focused hierarchy
- Gold accent highlights
- Flat UI (no heavy shadows or blur)
- Inspired by financial dashboards & global monitoring systems

## 🧰 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Tailwind CSS (CDN)
- Tone.js (audio engine)
- Lucide Icons

## 📁 Project Structure


project/
├── index.html
├── style.css
└── script.js


## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in browser
3. (Optional) Use Live Server for development

## ⌨️ Functional Modules

### World Clock
- Track multiple cities
- Analog + digital clock
- Real-time updates

### Stopwatch
- Start / Stop / Reset
- Lap recording

### Timer
- Countdown mode
- Auto trigger alert

### Alarm
- Custom label
- Sound selection
- Persistent storage (localStorage)

### Converter
- Convert time across timezones
